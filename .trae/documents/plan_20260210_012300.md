Saya akan mengubah aplikasi `index.html` yang ada menjadi proyek **Vite + React** yang modern dan menggunakan **Google Sheets** sebagai database (menggantikan Firebase).

### Strategi Migrasi
1.  **Frontend**: Menggunakan React (Vite) + Tailwind CSS.
2.  **Backend/Database**: Google Sheets + Google Apps Script (sebagai API perantara).
    *   *Kenapa Google Apps Script?* Ini adalah cara gratis dan aman untuk menghubungkan React ke Google Sheets tanpa mengekspos kunci rahasia di kode frontend. Anda tidak perlu server tambahan.

### Langkah-Langkah Implementasi

#### 1. Persiapan Proyek React (Saya akan lakukan ini)
*   Inisialisasi proyek Vite baru di direktori ini.
*   Instalasi **Tailwind CSS** dan konfigurasi agar sesuai dengan desain lama.
*   Instalasi library pendukung: `lucide-react` (ikon), `date-fns` (format tanggal).

#### 2. Konfigurasi Google Sheets (Anda perlu lakukan ini nanti)
Saya akan memberikan kode script, tapi Anda perlu:
*   Buat Google Sheet baru di akun Google Anda.
*   Buat 3 Tab: `Users`, `Checklist`, `Finance`.
*   Buka **Extensions > Apps Script**, tempel kode yang saya berikan.
*   Deploy sebagai **Web App** (Access: *Anyone*) untuk mendapatkan **API URL**.

#### 3. Pengembangan Fitur (Saya akan lakukan ini)
*   **Struktur Folder**: Membagi kode `index.html` yang panjang menjadi komponen React (`Sidebar`, `Dashboard`, `Operasional`, `Keuangan`).
*   **API Service**: Membuat file `src/services/api.js` untuk berkomunikasi dengan URL Google Apps Script Anda.
*   **State Management**: Mengganti Firebase `onSnapshot` (realtime) dengan mekanisme *polling* (refresh otomatis setiap interval tertentu) atau tombol refresh manual, karena Google Sheets tidak mendukung realtime push secara bawaan.
*   **Fitur**:
    *   **Login**: Simpan profil pengguna di LocalStorage dan Sheet `Users`.
    *   **Operasional**: Checkbox akan menyimpan status ke Sheet `Checklist`.
    *   **Keuangan**: Laporan akan dikirim (Append Row) ke Sheet `Finance`.

#### 4. Finalisasi
*   Menghapus file lama (`index.html`) setelah React siap.
*   Memastikan aplikasi berjalan lancar di browser.
